### SaaS Platform Plan

The major initiative is the [SaaS Auth Platform — Phased Build Plan](backlog.md#-saas-auth-platform--phased-build-plan) in the backlog. Six phases covering RS256+JWKS, Redis token denylist, multi-tenant org/app/user model, config portal, introspection, and revocation. See the backlog for full acceptance criteria.
